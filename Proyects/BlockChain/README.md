Proyectos de Blockchain y Criptografía
Esta sección del portafolio contiene diversas implementaciones de tecnología blockchain aplicadas a casos de uso reales. Todos los proyectos están diseñados para ejecutarse de manera local y ofrecen una interfaz visual interactiva.

Requisitos de Ejecución
Para el correcto funcionamiento de los archivos, es necesario contar con el siguiente entorno:

Dashboard de Streamlit: La interfaz de usuario de todos los proyectos se despliega mediante Streamlit.

Cartera de Bitcoins: Se requiere una cartera de bitcoins para las transacciones. Puede utilizar cualquier alternativa gratuita y conectarla al sistema.

Túnel de Ngrok: Se utiliza un túnel de Ngrok para establecer la conexión necesaria entre los componentes.

Base de Datos Local (MongoDB): Es indispensable contar con una instancia local de MongoDB para la construcción y persistencia de las bibliotecas y registros.

Proyectos Incluidos
Dentro de esta carpeta se encuentran diferentes aplicaciones de la tecnología de bloques, tales como:

Identificadores únicos: Sistema de gestión para una tienda (estilo Blockbuster) que utiliza blockchain para validar la identidad y autenticidad de los activos.

Sistema de Recompensas: Un modelo de lealtad para establecimientos (ej. una cafetería) basado en tarjetas de recompensas digitales seguras.

Sistema de Votaciones: Implementación de un proceso de votación transparente e inmutable mediante registros distribuidos.

Instrucciones de Uso
Los scripts pueden ejecutarse desde cualquier entorno de desarrollo estándar, como Visual Studio Code, Spyder o Google Colab.

Para poner en marcha cada proyecto:

Abra el archivo principal del proyecto deseado.

Verificación de dependencias: En la parte superior de cada archivo encontrará un comentario indicando el nombre de los archivos adicionales y bibliotecas que se requieren para su ejecución exitosa.

Ejecute el archivo siguiendo los comandos estándar de Python.
